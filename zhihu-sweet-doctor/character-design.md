## 一、男主完整人设

### 【基础档案】

| 项目 | 内容 |
|------|------|
| **姓名** | 顾衍之 |
| **年龄** | 32岁 |
| **职业** | 海城医科大学附属第一医院 神经外科副主任医师 |
| **身高** | 187cm |
| **外貌特征** | 俊朗立体，剑眉星目，鼻梁高挺，薄唇微抿，戴无框眼镜时更显禁欲气质；手术时换上白大褂和手术帽，眉眼间尽是专注与疏离；右眉骨有一道浅疤（医患纠纷留下，也是唯一一次"失控"） |

### 【性格画像】

**表层性格（他人眼中）**
- 高冷疏离，惜字如金
- 专业到近乎无情，手术台上是"机器"
- 不近女色，绯闻绝缘体
- 下属敬畏，患者爱慕又害怕

**深层性格（真实自我）**
- 内心极度缺乏安全感，害怕被抛弃
- 用"冷漠"保护自己，也推开别人
- 对感情极度笨拙，不知如何表达
- 责任感爆棚，把所有人的命看得比自己重

**性格关键词**: 外冷内热、自我压抑、情感钝感、责任执念

### 【家庭背景】

- **父亲**: 顾明远，神经外科泰斗级人物，学术权威，对儿子要求极严，从小就是"别人家的孩子"的标准答案。从未夸过儿子一句，导致顾衍之从小活在"不够好"的阴影里。
- **母亲**: 苏雅兰，文学教授，温柔但早逝（顾衍之18岁时因脑瘤去世，由儿子亲手签字放弃治疗）。这是顾衍之选择神外的根源——他要救回像母亲那样的人。
- **家庭氛围**: 严肃、压抑、学术至上。父亲再婚后（顾衍之24岁），他彻底把自己封闭起来。

**创伤根源**:
1. 母亲之死：亲手签字放弃治疗，18岁的他崩溃却不敢在父亲面前流泪
2. 前女友抛弃：28岁时，交往两年的女友说"你太冷了，我要的不是一台手术机器"，在手术日提分手，他没追
3. 职业阴影：见过太多生死，把"保持距离"当成自我保护

### 【职业成就】

- 30岁破格晋升副主任医师，神外"天才刀"
- 主刀手术成功率98.7%，业内有"顾氏一刀"之称
- 发表SCI论文23篇，获国家自然科学基金青年项目
- 暗地里资助了多位贫困患者，从不告诉任何人

### 【情感创伤】

**前女友**: 白薇，28岁离开他，嫌弃他"没有温度"

**三年前的林晓晓**: 
- 她是他带过的实习生中最让他印象深刻的学生
- 她的活泼、真诚、无所畏惧，像光照进他的世界
- 他心动了，但他不敢——"我才被说太冷，我不能耽误她"
- 表白那天，他用最冷漠的话拒绝："你不够格"、"我们不合适"、"找别人吧"

**拒绝后的自己**:
- 回到诊室，对着空房间说了一万遍"对不起"
- 在她离开后，连续三个月每晚失眠
- 把她的实习报告收藏在抽屉最深处
- 从此再没收过实习生

## 二、女主完整人设

### 【基础档案】

| 项目 | 内容 |
|------|------|
| **姓名** | 林晓晓 |
| **年龄** | 27岁（与男主相遇时24岁） |
| **职业** | 自由插画师 / 网络人气画手"晓晓画画" |
| **身高** | 165cm |
| **外貌特征** | 清秀灵动，杏眼微弯，笑起来有小梨涡，头发微卷，喜欢穿棉麻质地的衣服，整体气质温暖治愈；但眼神深处藏着不易察觉的疲惫和戒备；手腕内侧有一道浅疤（三年前的故事） |

### 【性格画像】

**表层性格（他人眼中）**
- 阳光开朗，笑容治愈
- 社交达人，朋友圈永远正能量
- 随遇而安，好像从不会生气
- 对谁都好，却没有特别亲密的人

**深层性格（真实自我）**
- 用笑容掩盖一切情绪
- 极度缺乏安全感，不敢真心付出
- 被拒绝后，再也没勇气主动
- 夜晚一个人时，是最真实的自己

**性格关键词**: 假性阳光、情感防御、内心孤寂、自我否定

### 【家庭背景】

- **父亲**: 林建国，高中语文老师，温和开明，支持女儿的一切选择
- **母亲**: 陈月华，护士长，坚强乐观，是林晓晓童年"笑容教育"的来源
- **弟弟**: 林小北，22岁，医学院学生，崇拜顾衍之（顾衍之不知道）

**家庭氛围**: 温暖、支持、但有期待。母亲希望她找份稳定工作，她选择自由职业让母亲担心。

**创伤根源**:
1. 大学时期的初恋背叛：谈了三年，毕业前夕发现对方劈腿，笑着的她说"没关系"，回宿舍哭了一整晚
2. 顾衍之的拒绝：鼓起全部勇气表白，被她敬仰的老师用最冷漠的方式拒绝，从此"热情"是她的保护色
3. 三年后的车祸：车祸时她第一个念头是"如果是他救我就好了"，然后嘲笑自己"还惦记呢？"

### 【职业成就】

- 网络画手"晓晓画画"，粉丝过百万
- 出版过两本治愈系绘本，《明天会更好》系列广受好评
- 自由职业三年，收入稳定，独自租住在医院附近（因为想离他近一点，又不敢靠近）
- 暗地里画了一套《那个医生》系列，主角是某位神外医生——从未发表

### 【情感创伤】

**初恋**: 大学时期谈了三年，毕业前夕对方劈腿。她笑着说"祝你幸福"，从此学会用笑保护自己。

**三年前的顾衍之**:
- 他是她实习时的带教老师，高冷但专业，偶尔流露的温柔让她心动
- 她写了整整三个月的告白信，表白那天手心全是汗
- 他的拒绝让她觉得自己"不够好"、"不配"
- 离开医院后，她删除了他的所有联系方式，却忍不住偷偷查他的消息

**三年里的自己**:
- 画了无数张他的素描，都锁在抽屉里
- 每次路过医院都会心跳加速，然后绕道走
- 假装已经"放下"，其实谁都没能靠近她

**车祸那天的想法**:
> "如果死前能再见他一面就好了……不，算了，他大概早不记得我了。"

## 四、CP感设定

### 【化学反应来源】

**互补性**:
| 男主 | 女主 |
|------|------|
| 冷 | 热 |
| 克制 | 自由 |
| 职业执念 | 人生松弛 |
| 情感钝感 | 情感敏感 |
| 不敢爱 | 不敢被爱 |

**相似性**:
- 都用"铠甲"保护自己（他的冷漠，她的笑容）
- 都在重要关系中受过伤
- 都渴望被坚定选择，却不敢主动

**宿命感**:
- 医患关系的必然交集
- 三年前她是他推开的人，三年后他必须追回她
- 他亲手救了她（手术），也要亲手医好她的心（追妻）

### 【互动模式设计】

**阶段 1：对立（重逢期）**
- 他想靠近，她装作不认识
- 他关心，她冷笑："顾医生，我们熟吗？"
- 他在病历上写满满一页，她翻到背面——"签名就行"

**阶段 2：拉锯（追妻期）**
- 他每天出现在病房，她当他是空气
- 他给她带早餐，她说"我在减肥"
- 他在手术室外守三天，她醒来第一句是"你怎么还没下班"

**阶段 3：破冰（暧昧期）**
- 她发烧他守夜，她迷糊中叫了他的名字
- 他教她康复训练，手心的温度让她心跳加速
- 她无意发现他收藏的照片，情绪崩溃

**阶段 4：告白（高潮期）**
- 他当众（科室、家人、前女友）承认："三年前是我错了"
- 她问他："你现在说这些，有意义吗？"
- 他回答："我想用余生，弥补三年的遗憾"

**阶段 5：暴甜（HE）**
- 他学做饭（一塌糊涂，但她都吃光）
- 他每天写一封情书（放她枕头下）
- 他在求婚时说："三年前我拒绝了你，今天你能不能拒绝我——拒绝我未来的每一天没有你？"

### 【成长互补点】

**男主成长弧**:
1. 学会表达情感——从"不说"到"说"
2. 学会面对过去——从"逃避"到"承认"
3. 学会主动追爱——从"推开"到"拉紧"

**女主成长弧**:
1. 学会相信——从"假性阳光"到"真实脆弱"
2. 学会表达不满——从"笑着接受"到"说出委屈"
3. 学会被爱——从"不配感"到"值得"

**互助攻点**:
- 他追她的过程，也是治愈自己"不敢爱"的过程
- 她学会被爱的过程，也是让他学会表达的过程
- 两人共同成长，最终成为彼此的"解药"

## 五、人物成长弧线

### 【男主：顾衍之】

```
起点：高冷、不敢爱、推开
  ↓
触发事件：女主车祸入院，他亲手救了她，也亲手触碰三年前的伤口
  ↓
挣扎：想弥补，却不知如何开口；想追，却习惯性"冷"
  ↓
转折：前女友出现，女主误会，他终于失控——在科室会议室当众告白
  ↓
成长：学会表达、学会道歉、学会"低到尘埃里"追一个人
  ↓
终点：治愈自己，也治愈她；不再是"高冷医生"，是"她一个人的顾衍之"
```

**关键节点台词**:
- 起点表白拒绝："你不够格，我们不合适。"（三年前）
- 挣扎期独白："我花了三年学会当医生，却没学会怎么对她说一句话。"
- 转折期告白："三年前我说你不合适。现在——现在我只希望，我还能合适你。"

### 【女主：林晓晓】

```
起点：活泼开朗（保护色）、不敢被爱、放下（假装）
  ↓
```

- 转折期崩溃："你藏着我照片三年……你知不知道我画了你三年？你知不知道我……"

## 七、追妻火葬场核心设计

### 【男主的"追妻障碍"清单】

| 障碍 | 来源 | 追妻过程中如何克服 |
|------|------|-------------------|
| 高冷 | 自我保护 | 学会表达——买早餐、送花、写情书、当众告白 |
| 不敢爱 | 前女友创伤 | 承认"我怕失去你，所以当初推开你" |
| 不会道歉 | 从未被肯定 | 学会说"对不起"，学会"低姿态" |
| 职业至上 | 母亲之死的阴影 | 把她的命放在第一位——辞职追妻、手术室外守三天 |

### 【女主的"追妻防线"】

| 防线 | 来源 | 如何被攻破 |
|------|------|-----------|
| 假性阳光 | 三年前被拒 | 在他面前终于崩溃、哭出来 |
| 不相信自己值得被爱 | 三年来的自我否定 | 他的死缠烂打、当众告白、三年的秘密 |
| 不敢再主动 | 表白失败的恐惧 | 他先低头，她才有勇气 |

### 【经典追妻场景预设】

1. **ICU外守夜**: 他连续三天在ICU外守着，护士说"顾医生您回去休息吧"，他说"她在里面，我睡不着"
2. **买早餐被拒**: 他买了她爱吃的早餐，她说"我在减肥"，他把早餐放在护士站："她饿了会来拿"
3. **当众告白**: 科室例会上，他突然说："各位，我有一件事要宣布——三年前我做了一件错事，现在我想弥补。"
4. **情书计划**: 他每天写一封情书，放她枕头下，连写60天

## 八、防御点：如何应对质疑

### 【Q1: 男主高冷是否OOC？】

**回应**: 
- 高冷不是性格，是创伤后的自我保护机制
- 背景：母亲18岁去世、前女友嫌弃"太冷"、职业习惯性压抑情感
- 证据：他收藏女主照片三年、偷偷关注她的社交媒体、在她表白后独自崩溃
- 高冷是"表面"，真实自我是"害怕失去所以不敢拥有"

### 【Q2: CP感是否足够？】

**回应**: 
- 互补性：冷 vs 热、克制 vs 自由、职业执念 vs 人生松弛
- 相似性：都用铠甲保护自己、都渴望被坚定选择
- 宿命感：医患关系必然交集、三年前推开的人必须追回
- 场景支撑：
  1. ICU外守夜：他用专业救她，用真心守她
  2. 康复训练：手牵手，心跳加速
  3. 发现秘密：他藏她照片三年，她画他三年——双向暗恋

### 【Q3: 追妻动机是否合理？】

**回应**: 
- 不是"突然追妻"，是"三年都在后悔"
- 证据：收藏照片、偷偷关注、三年没收实习生
- 触发事件：她出车祸，命悬一线，他终于意识到"失去才是真正的痛"
- 追妻不是因为他变了，是因为他终于敢面对自己的心
