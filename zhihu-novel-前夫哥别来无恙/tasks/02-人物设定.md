# 人物设定文档《前夫哥，别来无恙》

**作品类型**：双向成长 + 健康恋爱
**虐甜配比**：虐15% → 追妻50% → 甜宠35%
**核心定位**：不是"破镜重圆"，而是"她过得比他好"

---

## 一、女主人物小传

### 基础信息

| 项目 | 内容 |
|------|------|
| **姓名** | 林清婉 |
| **年龄** | 29岁（离婚时27岁） |
| **职业** | 独立花艺师（前金融分析师） |
| **外貌** | 168cm，清瘦但不单薄，骨架舒展；长发微卷，习惯低马尾；眉眼清淡，笑起来眼尾微微弯起，有种"不争不抢"的气质；穿衣风格极简，棉麻、莫兰迪色系，夏天常穿白色亚麻衬衫 |
| **声音** | 音色偏低，说话不紧不慢，习惯思考后再开口 |

### 性格特征（核心三特质）

**1. 温和但有边界**

- 表现：
  - 对谁都客气，但没人能真正走近她的"里圈"
  - 朋友约她从不拒绝，但聊到感情她会笑着说"那都是很久以前的事了"
  - 前夫的电话，她会接，语气正常，但绝不主动联系
  - 她的边界感不是冷漠，而是一种"我已经习惯了没有你"的从容

- 记忆点：
  - 别人问她"你还爱他吗"，她会认真地想一下，然后说"好像...不记得了"

**2. 享受独处，也享受生活**

- 表现：
  - 离婚后没有"涅槃重生"，也没有"郁郁寡欢"，她只是...开始学着做花艺
  - 发现自己喜欢植物的安静，于是辞了金融分析师的工作，开了一间小花店
  - 她的快乐来源很小：一束搭配满意的花、一杯咖啡的午后、一场一个人的电影
  - 她不再需要"被爱"来证明自己存在的意义

- 记忆点：
  - 她的社交媒体没有"单身宣言"，只有各种花的照片，和偶尔的咖啡、猫

**3. 有主见，但不强势**

- 表现：
  - 朋友让她"一定要反击前夫"，她会说"没必要，他过他的，我过我的"
  - 客人讨价还价，她微笑着说"这个价格我做不了，您可以去别家看看"
  - 她从不会大吵大闹，但心里非常清楚自己要什么

- 记忆点：
  - 前夫送花到店里，她会收下，转天把花放在店门口，贴了张纸条："免费赠送，今日新鲜"

### 背景故事

**成长经历**：
- 出身普通知识分子家庭，父母都是中学老师
- 从小被教育"要懂事、要体面、不要给别人添麻烦"
- 大学学金融，毕业后进投行，高强度工作让她习惯了"压抑情绪"
- 26岁遇到周瑾行，27岁结婚，29岁离婚——全程没有大吵大闹，只有平静的"我们分开吧"

**与男主的情感经历**：
- 相识：朋友聚会上，她是那个安静坐角落的人，他注意到她
- 追求：他很用力地追，她被"用力"打动——她以为那叫爱
- 婚姻：婚后发现他更爱他的"理想妻子"——温柔、顾家、不给他添麻烦
- 她努力成为那个"理想妻子"，却发现自己越来越不像自己
- 离婚：没有出轨、没有家暴，只是有一天她看着他，发现"我不想再这样了"
- 她说"我们分开吧"，他沉默了很久，说"好"

**离婚后的她**：
- 没有崩溃、没有逆袭，只是...松了一口气
- 辞职、学花艺、开店——她终于可以做自己喜欢的事
- 偶尔会想起他，但想起的是"那个好像不怎么开心的人"，而不是"我爱的人"

### 核心诉求

**表面诉求**：过好自己的生活
**深层诉求**：被尊重、被看见——不是作为"谁的妻子"，而是"她自己"

### 缺陷与脆弱点

**缺陷**：
- 过于习惯"独自消化"，导致她很难表达自己的需求
- 她的"云淡风轻"有时是一种保护色，她不擅长处理"被在意"的感觉

**脆弱点**：
- 她会想起婚后的很多个夜晚，一个人在厨房煮面，客厅里是他加班的背影
- 她害怕"回到过去"，所以下意识地和前夫保持距离
- 她其实很需要被爱，但她已经学会"不期待"

### 成长弧线

| 阶段 | 状态 | 关键事件 |
|------|------|---------|
| **婚姻期** | 压抑自己，努力成为"理想妻子" | 她以为这就是婚姻，直到她发现"我不想再这样了" |
| **离婚后（初期）** | 重新学习"做自己" | 辞职、学花艺，发现原来自己可以很开心 |
| **离婚后（中期）** | 真正的放下 | 不再刻意"忘记"，而是"想起来也没什么" |
| **重逢时** | 云淡风轻 | 她坐在他订婚宴的宾客席，举杯说"恭喜" |
| **追妻期** | 被打乱，但坚定 | 他的出现让她不习惯，她不想回到过去 |
| **最终** | 重新选择 | 不是"原谅"，而是"重新认识他之后，选择他" |

### 记忆点设计

1. **"免费赠送"的花**：
   - 他送的花，她第二天放在店门口免费送人
   - 不是报复，是"我已经不需要了"

2. **"不记得了"**：
   - 别人问她"你还爱他吗"，她会认真想一下，然后说"好像...不记得了"
   - 这句话贯穿全文，成为她的标志性台词

3. **订婚宴举杯**：
   - 重逢场景：她坐在宾客席，他站在台上，她举杯微笑说"恭喜"
   - 他的未婚妻注意到：这个女人的眼睛里，真的没有一丝波澜

## 二、男主人物小传

### 基础信息

| 项目 | 内容 |
|------|------|
| **姓名** | 周瑾行 |
| **年龄** | 32岁（离婚时30岁） |
| **职业** | 私募基金经理 |
| **外貌** | 183cm，肩宽腰窄，身材保持很好；短发干净利落，戴无框眼镜；眉眼锋利，但习惯性面无表情；穿深色西装很挺拔，穿休闲装时反而显得不自在 |
| **声音** | 低沉、语速偏快，谈公事时压迫感强，私下说话反而简短 |

### 性格特征（核心三特质）

**1. 目标导向，忽略过程**

- 表现：
  - 事业极度成功，因为他永远盯着目标，从不分心
  - 感情上也是——他追林清婉时很用力，追到了，就进入"维护模式"
  - 他以为婚姻是"任务完成"，却忘了婚姻需要持续经营
  - 离婚时他很震惊：他以为她很满意——因为她从不抱怨

- 记忆点：
  - 他的口头禅："这个事情，结果是什么？"

**2. 不擅长表达情感**

- 表现：
  - 爱一个人的方式是"给她最好的"——物质、资源、保障
  - 但他不会说"我爱你"，不会问"你今天开心吗"
  - 他以为她不需要，因为她从来不说
  - 他的情感世界是"黑箱"：他不知道怎么打开，也不知道怎么让她看到

- 记忆点：
  - 她离开后，他发现自己不知道她喜欢什么花

**3. 骄傲，但骄傲的根基正在崩塌**

- 表现：
  - 离婚后他告诉自己"她不适合我"，然后开始相亲
  - 订婚对象是门当户对的女孩，乖巧、温柔、很适合他——至少表面上
  - 直到他在订婚宴上看见林清婉，发现她眼神里没有他
  - 他的骄傲在那一刻出现了裂痕

### 背景故事

**成长经历**：
- 出身商人家庭，从小被教育"结果最重要"
- 母亲是全职太太，父亲忙于事业，家庭关系冷淡
- 他从小就知道：想要什么，要自己去拿
- 这让他成功，也让他不懂"关系"需要两个人

**与女主的情感经历**：
- 相识：被她的安静吸引——她不像他认识的其他女孩，聪明但锋利
- 追求：他追得很认真——鲜花、晚餐、接送，他以为这就是"用心"
- 婚姻：他以为自己做得很好——她从不抱怨，他把工资卡给她，买了她喜欢的房子
- 离婚：她说"我们分开吧"，他愣了很久——他不知道问题出在哪里
- 她说"我不想再这样了"，他问"哪样？"——她没有回答

**离婚后的他**：
- 失落，但告诉自己"是她不适合"
- 开始相亲，遇到苏婉（未婚妻）——门当户对，家长满意
- 他以为这就是"对的婚姻"——直到订婚宴上，他看见林清婉
- 她坐在宾客席，举杯说"恭喜"，眼睛里真的没有他

### 前期"瞎眼"动机

**核心动机：他以为"不抱怨"= "满意"**

| 他的认知 | 现实 |
|---------|------|
| 她从不抱怨，说明她很满意 | 她不抱怨，是因为她习惯了压抑 |
| 他把最好的给她（物质） | 她要的不是物质，是被看见 |
| 她很安静，很乖，很适合他 | 她很安静，是因为她发现"说没用" |
| 婚姻就是这样，平淡才是真的 | 她要的不是轰轰烈烈，而是"他在意她" |

**为什么会"瞎眼"**：
- 他从自己的家庭学到的婚姻模板：父亲忙事业，母亲顾家，相敬如宾
- 他以为"不吵架"就是好婚姻
- 他以为她在等他回家，其实她只是习惯了等

### 觉醒触发点

**第一层：订婚宴重逢**

- 他看见她，发现她比离婚时更好看——不是打扮，是整个人"亮了"
- 她举杯说"恭喜"，他发现她眼睛里真的没有他
- 这一刻，他第一次感到"失去"

**第二层：发现她的新生活**

- 他开始"无意间"路过她的花店
- 发现她的店很小，但很温馨，她在店里哼着歌
- 发现她有了新朋友，会一起喝咖啡、爬山、做陶艺
- 发现她的社交媒体上全是花、猫、她喜欢的各种小事
- 他发现：她的世界里，从来没有"等他回来"

**第三层：她对他的态度**

- 他开始"偶遇"她，她每次都客气地打招呼，然后自然地告别
- 他送花到她店里，她收下了——第二天放在店门口"免费赠送"
- 他试图约她吃饭，她说"谢谢，但最近有点忙"
- 他发现：她不是"假装不在意"，她是真的...不在意了

**第四层：终于明白**

- 他和一个朋友喝酒，朋友说："她不是不爱你了，她只是...学会了没有你"
- 这句话刺中了他
- 他开始回忆婚姻：她做的饭、她等他的背影、她想要聊天时他说的"我在忙"
- 他发现：他从来不知道她喜欢什么、她不开心的时候在想什么
- 他意识到：他不是"失去她"，而是"从来没有真正拥有过"

### 追妻态度转变轨迹

| 阶段 | 态度 | 行为 | 心理 |
|------|------|------|------|
| **离婚初期** | "是她不适合" | 相亲、订婚 | 否认、自我合理化 |
| **订婚宴重逢** | 被刺了一下 | 开始关注她 | 好奇、隐隐不安 |
| **发现她的新生活** | 不舒服 | "无意间"路过花店 | 对比心理：为什么她没有我也可以？ |
| **被"免费赠送"** | 受伤 | 更加频繁地出现 | 她不在乎？她怎么可以不在乎？ |
| **被拒绝约会** | 崩溃 | 开始反思 | 为什么？我做错了什么？ |
| **觉醒** | 痛苦 + 痴迷 | 放弃一切追她 | 我要她，我要重新认识她 |
| **追妻期** | 卑微 + 成长 | 各种追妻名场面 | 从"追回来"到"尊重她的选择" |

### 记忆点设计

1. **"我不知道她喜欢什么花"**：
   - 他送花，助手问他"什么花？"
   - 他愣了很久，发现自己不知道
   - 他开始偷偷学习花语、花艺

2. **"免费赠送"的反差**：
   - 他精心挑选的花，第二天出现在她店门口的纸箱里
   - 纸条上写着"免费赠送，今日新鲜"
   - 他站在店门口，看着那张纸条，站了很久

3. **"我在忙"回旋镖**：
   - 他终于约到她吃饭，聊到她以前想和他分享的事
   - 她说"那时候我总想和你说这些，但你在忙"
   - 他发现：他错过的是整整三年

## 三、人物关系图谱

### 核心关系时间线

```
【相识】→【追求】→【婚姻（3年）】→【离婚】→【他订婚】→【重逢】→【追妻】→【重新选择】
```

| 时间节点 | 他 | 她 | 关系状态 |
|---------|-----|-----|---------|
| **相识** | 被她吸引，决定追她 | 被他的"用心"打动 | 他主动，她接受 |
| **追求期** | 很用力地追 | 被感动，以为这就是爱 | 他热烈，她被动但心动 |
| **婚姻** | 进入"维护模式"，以为她满意 | 努力成为"理想妻子"，逐渐失去自己 | 他忽略，她压抑 |
| **离婚** | 震惊，但接受 | 平静，但内心松了一口气 | 他困惑，她解脱 |
| **离婚后（他）** | 相亲，遇到未婚妻 | — | 他试图向前走 |
| **离婚后（她）** | — | 辞职、学花艺、开店 | 她开始做自己 |
| **订婚宴重逢** | 被她刺痛 | 举杯说"恭喜"，眼睛里没有他 | 他开始在意，她云淡风轻 |
| **追妻期** | 放弃一切追她 | 被打乱，但不想回去 | 他卑微，她坚定 |
| **最终** | 学会尊重她 | 重新认识他，做出选择 | 双方成长，重新开始 |

### 重要配角

**1. 苏婉（未婚妻）—— 他成长的"镜子"**

| 项目 | 内容 |
|------|------|
| **身份** | 订婚对象，门当户对的女孩 |
| **性格** | 乖巧、温柔、识大体，像极了婚姻初期的林清婉 |
| **作用** | 让他意识到：他追求的"理想妻子"，其实是他不快乐的原因 |
| **关键情节** | 订婚宴上，苏婉注意到林清婉，问她"那是谁"，他沉默了很久，说"一个...故人" |

**2. 陆小小（花店合伙人）—— 她的"新世界"**

| 项目 | 内容 |
|------|------|
| **身份** | 林清婉的花店合伙人，比她小3岁 |
| **性格** | 活泼、直率、爱吐槽，是林清婉离婚后最好的朋友 |
| **作用** | 陪伴、吐槽、助攻，让林清婉的新生活"有声音" |
| **关键情节** | 她是第一个说"他配不上你"的人，也是第一个发现林清婉"好像被打乱了"的人 |

**3. 周父周母 —— 他的"婚姻模板"**

| 项目 | 内容 |
|------|------|
| **身份** | 周瑾行的父母 |
| **婚姻** | 相敬如宾的冷淡婚姻 |
| **作用** | 让他理解自己"瞎眼"的根源——他从父母那里学到的婚姻模板，本就是错的 |
| **关键情节** | 他去看父母，发现他们各做各的事，互不打扰，突然意识到：这就是他婚姻的样子 |

### 人物冲突点

**核心冲突**：
- 他：想要追回她，证明自己可以改变
- 她：不想回去，她已经学会了没有他

**关键矛盾**：
1. 他以为她在"假装不在意" vs 她真的不在意了
2. 他想"追回来" vs 她已经"向前走了"
3. 他的"用力追" vs 她的"需要空间"

**化解路径**：
- 他从"追回来"变成"尊重她的选择"
- 她从"不想回去"变成"重新认识他"
- 最终：不是回到过去，而是重新开始

## 四、人设风险预警

### 风险1：女主太"冷淡"，读者难以代入

**问题**：云淡风轻容易变成"冷血"，读者会觉得她不够爱男主。

**规避方案**：
- 加入"脆弱点"：她会想起婚姻中的某些瞬间，她不习惯"被在意"
- 追妻后期，她开始被打乱：她发现自己不习惯了
- 强调：她的"云淡风轻"是保护色，不是没有感情

### 风险2：男主"追妻"太顺利

**问题**：如果追妻太顺利，"健康恋爱"会变成"她轻易原谅"。

**规避方案**：
- 她不轻易接受，她需要看到他的真正改变
- 他的成长是：从"追回来"到"尊重她的选择"
- 最终复合不是"原谅"，是"重新认识后的选择"

### 风险3：未婚妻工具人

**问题**：苏婉容易被写成脸谱化"恶毒女配"。

**规避方案**：
- 苏婉不是坏人，她只是"不适合他"
- 她的温柔乖巧，恰好是周瑾行婚姻失败的原因——他需要的是"真实的伴侣"，不是"理想妻子"
- 苏婉主动解除婚约，因为她发现：他看她的眼神，和看林清婉不一样

### 风险4：复合逻辑不够强

**问题**：读者会问"为什么要复合？她不是过得很好吗？"

**规避方案**：
- 复合不是"回去"，是"重新开始"
- 她需要看到的不是"他变了"，而是"他愿意认识真正的她"
- 他需要证明的不是"我爱她"，而是"我可以尊重她的选择"
- 最终决定权在她：她选择他，不是因为"原谅"，是因为"值得"

## 五、核心场景预埋

### 场景1：订婚宴重逢（开篇钩子）

- 她坐在宾客席，他站在台上
- 她举杯说"恭喜"，眼睛里真的没有他
- 苏婉注意到，问她"那是谁"，他沉默

### 场景2：花店"偶遇"（追妻开始）

- 他开始"路过"她的花店
- 发现她在店里哼着歌，她在微笑
- 他发现：她的世界里，没有他也可以很快乐

### 场景3："免费赠送"（追妻名场面）

- 他精心挑选的花，第二天出现在店门口
- 纸条："免费赠送，今日新鲜"
- 他站在门口，看着那张纸条，站了很久

### 场景4：醉酒后的话（追妻高潮）

- 他喝醉了，站在她店门口
- 她出来，他问她："你怎么可以...不在乎？"
- 她沉默了很久，说："因为你从来没有...真正看见过我"

### 场景5：她被打乱（转折）

- 他开始频繁出现，但她不想回到过去
- 她对陆小说："我不想...再回到那种生活"
- 陆小小说："那就别回去，看看他能不能...跟上你"

### 场景6：他放弃"追回来"（成长）

- 他终于明白：他不能"追回来"，他需要"重新认识她"
- 他开始问：你想要什么？你喜欢什么？你不开心的时候在想什么？
- 她第一次发现：他好像...真的不一样了

### 场景7：她做出选择（结局）

- 他对她说："我不会让你回去，我只是想...认识现在的你"
- 她看着他，第一次认真地想：我要不要...再试一次？
- 她说："好，但这一次...你要认识的是我，不是你的'理想妻子'"
